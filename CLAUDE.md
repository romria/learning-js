# AGENTS.md

Shared guidance for AI coding agents working with this repository.

## Purpose

This repository is a curated list of **free** web development learning resources, organized by topic (HTML/CSS, JavaScript, TypeScript, React, Node.js, Git, Databases) and format (Coding, Video, Text, Mixed). All content lives in `README.md`.

## Contributing

When adding resources:
- Resources must be free
- Place each resource under the correct topic section and format subsection
- Use the existing link format: `* [Title (Platform/Author)](URL)`
- Keep sections ordered consistently: Coding → Video → Text → Mixed
